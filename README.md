# Ардчилсан Нам — Вэбсайт + Админ панель

Бүрэн ажиллагаатай вэбсайт: **Node.js + Express backend**, **SQLite database**, **vanilla HTML/CSS/JS frontend**, мөн **мэдээ, гишүүд, header (hero) зургуудыг** удирдах боломжтой **админ панель**.

## ✨ Боломжууд

- 🏠 Олон хуудастай нийтэд харагдах сайт (Эхлэл, Намын тухай, Үзэл баримтлал, Гишүүд, Мэдээ, Санхүү, Салбар гэх мэт)
- 🖼️ **Header дээрх эргэлддэг зургууд (hero slider)** — админ панелаас зураг **upload** хийж, гарчиг/тайлбар/товчнуудыг тохируулна
- 📰 **Мэдээ удирдах** — Мэдээ / Үйл явдал / Мэдэгдэл гэсэн 3 төрөл, зурагтай эсвэл эможитой, нэмэх/засах/устгах
- 👥 **Гишүүд удирдах** — Намын удирдлага / НБД / УИХ / НИТХ гэсэн бүлгүүдээр, профайл зураг **болон намтар (description)**-той. Гишүүний картан дээр дарахад намтрыг нь дэлгэрэнгүй хуудаснаас харна
- 🔐 Админ нэвтрэх систем (нэвтрэх нэр/нууц үг, сессээр хамгаалагдсан)
- 💾 Бүх өгөгдөл **SQLite** өгөгдлийн санд хадгалагдана (`server/data.db`) — файл устаж эвдрэхгүй
- 📤 Зураг upload хийхэд `public/uploads/` фолдерт хадгалагдаж, тухайн мэдээ/гишүүн/слайдыг устгахад зураг нь мөн автоматаар устдаг

## 🚀 Хэрхэн ажиллуулах вэ

**Шаардлага:** Node.js v22.5+ (вэб сайт `node:sqlite` гэсэн Node.js-д суурилагдсан sqlite ашигладаг тул native build шаардахгүй).

```bash
# 1. Хавтас руу орох
cd democracy-site

# 2. Шаардлагатай сангуудыг суулгах
npm install

# 3. (Заавал биш) .env файл үүсгэх — session secret-ээ солих санал болгож байна
cp .env.example .env

# 4. Серверийг асаах
npm start
```

Сервер амжилттай асвал:
```
✅ Server running at http://localhost:3000
   Admin panel:  http://localhost:3000/admin/
```

- Нийтэд харагдах сайт: **http://localhost:3000**
- Админ панель: **http://localhost:3000/admin/**

```
Нэвтрэх нэр: admin
Нууц үг:     admin123
```

⚠️ **Маш чухал:** Эхний нэвтрэлтийн дараа **Тохиргоо** хэсгээс нууц үгээ нэн даруй солино уу!

## 🖼️ Header зурагнуудыг хэрхэн солих вэ

1. Админ панель руу нэвтэрнэ (`/admin/`)
2. Зүүн талын цэснээс **"🖼️ Header зурагнууд"** дээр дарна
3. **"+ Шинэ слайд"** товч дарж зураг, гарчиг, тайлбар, товчнуудаа бөглөнө
   - Зураг оруулахгүй бол градиент өнгөөр харагдана (`gradient` талбарт жишээ: `linear-gradient(135deg,#001f5c,#1565c0)`)
4. Хадгалмагц нүүр хуудасны header дээр шууд харагдана (хуудсыг дахин ачаалбал)
5. Олон слайд нэмбэл автоматаар 5.5 секунд тутамд эргэлдэнэ

## 📦 Деплой хийх (production)

Энэхүү сервер нь statеless биш — файлын систем дээр SQLite database болон upload хийсэн зургуудыг хадгалдаг тул:

- **VPS** (DigitalOcean, Hetzner, гэх мэт) дээр PM2-той ажиллуулахыг зөвлөж байна:
  ```bash
  npm install -g pm2
  pm2 start server/server.js --name democracy-site
  ```
- **Render.com / Railway.app** мэт сервис дээр deploy хийж болно, гэхдээ "persistent disk" тохиргоог `server/data.db` болон `public/uploads/` зам дээр заавал асаана уу (үгүй бол deploy бүрд өгөгдөл устана)
- Production-д ашиглахын өмнө `.env` дотор `SESSION_SECRET`-ийг урт санамсаргүй мөрөөр солино уу
- HTTPS ашиглаж байгаа бол `server/server.js` доtorх session cookie тохиргоонд `secure: true` нэмж болно

## 🛠️ Технологи

| Хэсэг | Технологи |
|---|---|
| Backend | Node.js, Express |
| Database | SQLite (Node.js-ийн built-in `node:sqlite` модуль) |
| Frontend | Vanilla HTML, CSS, JavaScript (framework шаардахгүй) |
| Файл upload | Multer |
| Нэвтрэлт | express-session + bcryptjs |
